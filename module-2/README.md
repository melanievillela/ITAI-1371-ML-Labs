# Module 2: Tools Used in Machine Learning

Get hands-on experience with the essential development tools, libraries, and workflows for machine learning projects.

## Topics Covered

- **Development Environments**
  - Google Colab: Cloud-based notebooks (no installation required)
  - Jupyter Notebook: Local interactive computing
  - VS Code: Professional code editor with Jupyter support

- **Python Libraries for ML**
  - **Pandas**: Data manipulation and analysis
  - **NumPy**: Numerical computing and arrays
  - **Matplotlib**: Data visualization and plotting
  - **scikit-learn**: Machine learning algorithms

- **Data Exploration Fundamentals**
  - Loading datasets
  - Understanding data structure and shape
  - Exploratory data analysis (EDA)
  - Basic statistical operations

- **Version Control & Collaboration**
  - GitHub for storing and sharing projects
  - Repository creation and management
  - Professional documentation with Markdown

## Lab Exercise

**Module_02_Lab_Exercise_1.ipynb** — Get started with the Iris dataset. Learn library imports, data loading, exploration, and visualization in an interactive environment.

### Key Skills
- Import and configure machine learning libraries
- Load and explore datasets with pandas
- Perform basic statistical analysis on data
- Create informative visualizations with Matplotlib
- Write professional documentation using Markdown
- Initialize GitHub repositories for ML projects

```bash
# Option 1: Google Colab (recommended for beginners)
```

## What You'll Build

Using the Iris dataset, you'll:
- Load data and inspect its structure (150 samples, 4 features, 3 classes)
- Calculate basic statistics (mean, standard deviation) with NumPy
- Visualize relationships between sepal length and width by species
- Practice groupby operations and data aggregation with pandas
