# Module 3: Machine Learning Workflow and Types of Learning

Master the complete machine learning workflow and understand the three main paradigms of machine learning.

## Topics Covered

- **Types of Machine Learning**
  - Supervised Learning: Classification and regression
  - Unsupervised Learning: Clustering and dimensionality reduction
  - Reinforcement Learning: Learning through rewards and penalties

- **Complete ML Workflow** (6-step process)
  - Data preparation and exploration
  - Feature selection
  - Data splitting (train/test)
  - Model training
  - Model evaluation and metrics
  - Model interpretation

- **Classification Models**
  - Logistic Regression
  - Decision Trees
  - Performance evaluation with confusion matrices and classification reports

- **Data Types in Machine Learning**
  - Numerical (continuous and discrete)
  - Categorical (nominal and ordinal)
  - Text and boolean data
  - Preprocessing for different data types

## Core Concepts

### The ML Workflow
```
Problem Definition → Data Preparation → Feature Selection → 
Data Splitting → Model Training → Evaluation → Interpretation
```

### When to Use Each Learning Type
- **Supervised**: When you have labeled data and need predictions
- **Unsupervised**: When you want to discover patterns without labels
- **Reinforcement**: When learning through interaction and feedback is needed
